## 产品打包

### 1. 构建准备
```bash
# 安装构建依赖
npm install

# 运行测试
node test-bookmark-manager.js

# 检查代码质量
npm run lint  # (如果有配置)
```

### 2. 构建发布版本
```bash
# 构建所有平台
npm run build

# 或分别构建
npm run build-mac    # macOS 版本
npm run build-win    # Windows 版本
```
