# 🔄 Atlas重启按钮已移除

## 移除原因
Atlas重启按钮在应用中无法正常工作，虽然命令行测试正常，但在Electron应用的IPC通信中存在问题。为了避免用户困惑，暂时移除此功能。

## 当前解决方案
**手动重启Atlas浏览器**：
1. 完全关闭Atlas浏览器
2. 重新打开Atlas浏览器
3. 查看同步结果

## 替代方法

### macOS用户
```bash
# 关闭Atlas
osascript -e 'tell application "ChatGPT Atlas" to quit'

# 启动Atlas
open -a "ChatGPT Atlas"
```

### 或者直接操作
1. 使用 `Cmd+Q` 完全退出Atlas
2. 从应用程序文件夹重新打开Atlas
