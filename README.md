## ğŸ‡¬ğŸ‡§ English

**Echo Bookmark Sync** is a free, non-commercial open-source tool designed to bridge the gap between your traditional browsing experience and your AI workflow. It ensures your bookmarks are always in sync between **Google Chrome** and the **ChatGPT Atlas** macOS application.

### âœ¨ Key Features

*   **Bidirectional Sync**: Changes in Chrome reflect in Atlas, and vice versa.
*   **Smart Detection**: Automatically finds your browser bookmark paths on startup.
*   **Echo Logging**: Detailed, transparent logs showing exactly what was added, deleted, or renamed. "Echo" means instant feedback!
*   **Privacy First**: All data processing is done locally on your machine. No data is sent to any external servers.
*   **Safe & Non-Destructive**: Merges bookmarks intelligently based on timestamps to prevent data loss.

### ğŸš€ Installation & Usage

1.  Download the latest `.dmg` from the [Releases](https://github.com/yourusername/echo-bookmark-sync/releases) page.
2.  Install `Echo Bookmark Sync.app` to your Applications folder.
3.  Launch the app. It will auto-detect your Chrome and Atlas bookmark paths.
4.  Click **"Start Auto Sync"** for real-time monitoring, or use **"Manual Sync"** for a one-time update.
